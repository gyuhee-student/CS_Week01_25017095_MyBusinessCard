## 구현 시 어려웠던 점
- '사진 변경' 버튼을 눌렀을 때 해당 기능이 실행되도록 작업하는 것에서 어려움이 있었다. 먼저 정확히 어느 디렉터리에 새로운 사진 파일을 추가해야 하는지 알아야 했고, 사진 파일 추가했더라도 프로그램이 해당 사진 파일을 인식하지 못하는 문제가 있었다. 하지만 리소스에 대해서 이해하고, 리소스 파일을 추가하는 여러 방법을 시도하여 해결할 수 있었다.
